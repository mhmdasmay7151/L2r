# Security Policy

## Supported Versions

| Version | Status |
|--------:|--------|
| 1.x     | Supported |
| <1.0    | End-of-life |

## Reporting a Vulnerability
Do **not** open public issues. Contact: https://irradi.ato.rs

## Coordinated Disclosure
Default 90 days unless actively exploited.

## Local/Dev Hardening
Local/dev only; do not expose publicly without hardening.
